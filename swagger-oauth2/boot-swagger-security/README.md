# How to run

From the project root:
```bash
mvn install
mvn spring-boot:run
```

Get to the following url: localhost:8080/swagger-ui.html
When prompted to enter your user credentials, enter:
* username: javainuse
* password: javainuse

Also if we go to http://localhost:8080/v3/api-docs.
